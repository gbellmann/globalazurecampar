Global Azure Bootcamp 2019

27 de Abril - 8:30hs

Argentina

¿Qué es el Global Azure Bootcamp?

En Abril de 2013 se organizó el primer Global Azure Bootcamp en más de 134 localizaciones alrededor del mundo. En Argentina, el evento se realiza desde el año 2014 y reune cada año a más de 80 profesionales de IT junto con los máximos expertos del país sobre la nube de Microsoft para compartir experiencias, conocimiento, hands-on-labs y un día completo conociendo un poco más sobre Cloud Computing.

¿Dónde se realiza?
Microsoft de Argentina. Bouchard 710, Ciudad de Buenos Aires.
Inscripción
Preguntas frecuentes

¿Necesito llevar algo preparado?

Aprovecharás mejor el evento si te traes tu propia notebook preparada con Visual Studio 2017 o superior con las herramientas de Cloud instaladas, Azure PowerShell y la Azure CLI. Es mejor que lleves instalado dicho software para que no perder tiempo durante el evento.

También necesitarás una subscripción de Microsoft Azure. Hay muchas opciones incluyendo una trial de 30 días totalmente gratuita o los Azure passes que entregaremos durante el evento (de 80 dólares para gastar en un mes).

¿El evento es para principiantes?

Sí y no. Nos centraremos en una serie de sesiones y hands on labs que apuntan a un nivel intermedio, pero también profundizaremos en escenarios reales, zonas de preguntas y respuestas, etc.
